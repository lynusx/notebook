# 控制流

控制流（Control Flow）是程序执行的顺序和路径，决定了代码语句的执行次序。

## 第1章 循环语句

### 1.1 `for` 循环

`for` 循环用于已知迭代次数的场景。

#### 1.1.1 标准`for` 循环

```dart
void main() {
  // 打印 1 到 5
  for (int i = 1; i <= 5; i++) {
    print('第 $i 次循环');
  }
}
```

#### 1.1.2 `for-in` 循环

用于遍历实现 `Iterable`  接口的对象（如 `List` 或`Set` ）。

```dart
void main() {
  List<String> fruits = ['苹果', '香蕉', '橙子'];

  // 使用索引遍历
  for (int i = 0; i < fruits.length; i++) {
    print('$i: ${fruits[i]}');
  }

  // 使用 for-in 遍历（更推荐）
  for (var fruit in fruits) {
    print(fruit);
  }
  
  // 使用 final（推荐）
  for (final fruit in fruits) {
    print(fruit);
  }
}
```

#### 1.1.3 `forEach()`

对于可迭代类，还可以使用 `forEach()` 进行迭代。

```dart
void main() {
  List<int> numbers = [1, 2, 3, 4, 5];

  // 使用 forEach
  numbers.forEach((number) {
    print(number * 2);
  });

  // 使用箭头函数
  numbers.forEach((n) => print(n * 2));
}
```

#### 1.1.4 `for` 循环中的模式匹配

```dart
void main() {
  // 结构 Map 条目
  final hist = {'a': 23, 'b': 100};

  for (final MapEntry(key: key, value: count) in hist.entries) {
    print('$key occurred $count times');
  }

  // 简写形式
  for (final MapEntry(:key, value: count) in hist.entries) {
    print('$key occurred $count times');
  }
}
```

### 1.2 `while` 与`do-while`

#### 1.2.1 `while` 循环

`while` 循环在条件为 true 时持续执行，先判断条件再执行，条件不满足时一次都不执行。

```dart
void main() {
  int count = 0;
  
  // 先检查条件再执行
  while (count < 5) {
    print('计数: $count');
    count++;
  }
  
  // while：条件不满足时一次都不执行
  int a = 10;
  while (a < 5) {
    print('这行不会输出');
  }
}
```

#### 1.2.2 `do-while` 循环

`do-while` 循环先执行一次，再判断条件，确保至少执行一次。

```dart
void main() {
  int count = 0;
  
  // 先执行一次再检查条件
  do {
    print('计数: $count');
    count++;
  } while (count < 5);
  
  // do-while：至少执行一次
  int b = 10;
  do {
    print('这行会输出一次');
  } while (b < 5);
}
```

### 1.3 循环流程控制

#### 1.3.1 `break` 

`break` 用于立即终止循环。

```dart
void main() {
  // 查找第一个偶数
  List<int> numbers = [1, 3, 5, 8, 9, 10];

  for (var num in numbers) {
    if (num % 2 == 0) {
      print('找到第一个偶数: $num');
      break; // 找到后立即退出
    }
  }
}
```

#### 1.3.2 `continue` 

`continue` 用于跳过当前迭代，继续下一次循环。

```dart
void main() {
  // 打印所有奇数
  for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
      continue; // 跳过偶数
    }
    print(i); // 只输出奇数
  }
}
```

#### 1.3.3 标签

标签是一个以冒号（ `labelName:` ）结尾的标识符，你可以将其放在语句之前以创建带标签的语句。

```dart
void main() {
  // 使用标签跳出外层循环
  outerLoop:
  for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
      if (i == 2 && j == 2) {
        break outerLoop; // 跳出外层循环
      }
      print('i=$i, j=$j');
    }
  }
}
```

## 第2章 分支语句

### 2.1 `if-else` 

Dart 的条件表达式必须是 `bool` 类型，不会隐式转换非布尔值为布尔值。

```dart
void main() {
  int score = 85;
  
  if (score >= 90) {
    print('优秀');
  } else if (score >= 80) {
    print('良好');
  } else if (score >= 60) {
    print('及格');
  } else {
    print('不及格');
  }
```

### 2.2 `switch-case`

 用于多分支条件判断，比多个 `if-else` 更清晰。

#### 2.2.1 基本用法

Dart 3.0+ 中，非空的 `switch` 语句必须穷尽所有可能情况，或包含 `default` 子句。

```dart
void main() {
  String command = 'start';
  
  switch (command) {
    case 'start':
      print('启动应用');
      break;
    case 'stop':
      print('停止应用');
      break;
    case 'restart':
      print('重启应用');
      break;
    default:
      print('未知命令: $command');
  }
}
```

#### 2.2.2 `switch` 表达式

`switch` 表达式可以返回值，语法更简洁。

```dart
void main() {
  // 使用 switch 语句
  // Where slash, star, comma, semicolon, etc., are constant variables...
  switch (charCode) {
    case slash || star || plus || minus: // Logical-or pattern
      token = operator(charCode);
    case comma || semicolon: // Logical-or pattern
      token = punctuation(charCode);
    case >= digit0 && <= digit9: // Relational and logical-and patterns
      token = number();
    default:
      throw FormatException('Invalid');
  }
  
  // 使用 switch 表达式
  token = switch (charCode) {
    slash || star || plus || minus => operator(charCode),
    comma || semicolon => punctuation(charCode),
    >= digit0 && <= digit9 => number(),
    _ => throw FormatException('Invalid'),
  };
}
```

#### 2.2.3 穷尽性检查

Dart 分析器确保 `switch` 处理所有可能情况。

```dart
// 布尔类型（穷尽）
void main() {
  bool b = true;
  switch (b) {
    // ✓ 穷尽：只有 true 和 false 两种情况
    case true:
      print('yes');
    case false:
      print('no');
  }
}

// 可空类型（需要处理 null）
void main() {
  bool? b;
  switch (b) {
    case true:
      print('yes');
    case false:
      print('no');
    case null: // 必须处理 null
      print('null');
  }
}
```

#### 2.2.4 守卫子句

使用 `when` 关键字添加额外条件。

```dart
switch (pair) {
  case (int a, int b) when a > b:
    print('First element greater');
  case (int a, int b):
    print('First element not greater');
  default:
    print('Not an int pair');
}
```

#### 2.2.5 逻辑或

```dart
void main() {
  switch (platform) {
    case TargetPlatform.android:
    case TargetPlatform.fuchsia:
      return buildMaterialApp();  // android、fuchsia 共享逻辑
    case TargetPlatform.iOS:
      return buildCupertinoApp(); // ios 独享逻辑
    case TargetPlatform.macOS:
    case TargetPlatform.windows:
    case TargetPlatform.linux:
      return buildDesktopApp();   // macos、windows、linusx共享逻辑
  }
}
```

### 2.3 条件表达式

#### 2.3.1 三元运算符

```dart
void main() {
  int a = 10;
  int b = 20;
  
  // 传统写法
  int max;
  if (a > b) {
    max = a;
  } else {
    max = b;
  }
  
  // 三元运算符
  int max2 = a > b ? a : b;
  
  print('最大值是: $max2');  // 输出: 最大值是: 20
}
```

#### 2.3.2 空值判断运算符 ??

```dart
void main() {
  String? name;
  
  // 如果 name 为 null，使用默认值
  String displayName = name ?? '匿名用户';
  print(displayName);  // 输出: 匿名用户
  
  // 级联使用
  String? email;
  String result = email ?? name ?? '无联系方式';
  print(result);  // 输出: 无联系方式
}
```

#### 2.3.3 ??= 赋值运算符

```dart
void main() {
  int? count;
  
  // 仅在 count 为 null 时赋值
  count ??= 0;
  count ??= 100;  // 不会执行，因为 count 已有值
  
  print(count);  // 输出: 0
}
```

## 第3章 错误处理

### 3.1 `throw` 

用于抛出或者引发一个错误，Dart 可以抛出任何非 null 对象。

```dart
// 抛出异常对象
throw FormatException('Expected at least 1 section');

// 抛出任意对象
throw 'Out of llamas!';

// 在表达式中抛出异常
void distanceTo(Point other) => throw UnimplementedError();
```

### 3.2 `try-catch` 

#### 3.2.1 捕获异常

`catch（）` 接受两个参数，第一个是错误对象，第二个是堆栈信息 [StackTrace](https://api.dart.cn/dart-core/StackTrace-class.html) 。

```dart
void main() {
  try {
    breedMoreLlamas();
  } on OutOfLlamasException { // 捕获特定异常
    // A specific exception
    buyMoreLlamas();
  } on Exception catch (e) {  // 捕获并处理特定异常
    // Anything else that is an exception
    print('Unknown exception: $e');
  } catch (e, s) {
  	print('Exception details:\n $e');
  	print('Stack trace:\n $s');
	}
}
```

#### 3.2.2 `rethrow` 

将异常重新抛出，保留原始堆栈跟踪：

```dart
void main() {
  try {
    dynamic foo = true;
    print(foo++); // 运行时错误
  } catch (e) {
    print('misbehave() partially handled ${e.runtimeType}.');
    rethrow; // 允许调用者看到异常
  }
}
```

#### 3.2.3 `final` 

无论是否发生异常都会执行：

```dart
// 结合 catch 和 finally
try {
  breedMoreLlamas();
} catch (e) {
  print('Error: $e');
} finally {
  cleanLlamaStalls();
}
```

#### 3.2.3 自定义异常

```dart
class OutOfLlamasException implements Exception {
  final String message;
  OutOfLlamasException(this.message);

  @override
  String toString() => 'OutOfLlamasException: $message';
}

void breedMoreLlamas() {
  throw OutOfLlamasException('Not enough llamas!');
}
```

### 3.3 assert

开发阶段如果条件为`false` ，可以使用断言来中断正常执行。

```dart
// 断言语句：assert(<condition>, <optionalMessage>);

// 基本用法
assert(
  urlString.startsWith('https'),
  'URL ($urlString) should start with "https".',
);


// 常用场景
class Rectangle {
  final double width;
  final double height;

  Rectangle(this.width, this.height) {
    // 构造函数中验证参数
    assert(width > 0, '宽度必须大于0');
    assert(height > 0, '高度必须大于0');
  }

  double get area => width * height;
}

void main() {
  var rect = Rectangle(10, 20);
  print('面积: ${rect.area}');

  // 这会触发断言错误
  //   var invalid = Rectangle(-5, 10);
}

// 断言条件
void processData(List<int> data) {
  // 验证前置条件
  assert(data.isNotEmpty, '数据不能为空');
  assert(data.every((e) => e >= 0), '数据不能包含负数');

  // 处理数据
  int sum = data.reduce((a, b) => a + b);
  print('总和: $sum');
}

void main() {
  processData([1, 2, 3, 4, 5]); // 正常
  //   processData([]); // 断言失败
  //   processData([1, -2, 3]); // 断言失败
}
```

## 第5章 示例

### 5.1 用户权限系统

```dart
enum UserRole { guest, user, moderator, admin }

class User {
  final String name;
  final UserRole role;
  final bool isActive;
  
  User(this.name, this.role, {this.isActive = true});
}

void checkPermission(User user, String action) {
  // 验证用户状态
  assert(user.isActive, '用户账户未激活');
  
  switch (user.role) {
    case UserRole.guest:
      if (action == 'view') {
        print('${user.name} 可以查看内容');
      } else {
        print('${user.name} 权限不足');
      }
      break;
      
    case UserRole.user:
      if (action == 'view' || action == 'comment') {
        print('${user.name} 可以$action');
      } else {
        print('${user.name} 权限不足');
      }
      break;
      
    case UserRole.moderator:
      if (['view', 'comment', 'edit', 'delete'].contains(action)) {
        print('${user.name} 可以$action');
      } else {
        print('${user.name} 权限不足');
      }
      break;
      
    case UserRole.admin:
      print('${user.name} 拥有所有权限，可以$action');
      break;
  }
}

void main() {
  var admin = User('张三', UserRole.admin);
  var guest = User('李四', UserRole.guest);
  
  checkPermission(admin, 'delete');  // 张三 拥有所有权限，可以delete
  checkPermission(guest, 'view');    // 李四 可以查看内容
  checkPermission(guest, 'delete');  // 李四 权限不足
}
```

### 5.2 购物车计算

```dart
class CartItem {
  final String name;
  final double price;
  final int quantity;

  CartItem(this.name, this.price, this.quantity) {
    assert(price > 0, '价格必须大于0');
    assert(quantity > 0, '数量必须大于0');
  }

  double get subtotal => price * quantity;
}

class ShoppingCart {
  final List<CartItem> items = [];

  void addItem(CartItem item) {
    items.add(item);
  }

  double calculateTotal() {
    double total = 0;

    for (var item in items) {
      total += item.subtotal;
    }

    return total;
  }

  void printReceipt() {
    print('========== 购物小票 ==========');

    int index = 1;
    for (var item in items) {
      print('$index. ${item.name}');
      print('   单价: ¥${item.price} × ${item.quantity} = ¥${item.subtotal}');
      index++;
    }

    print('------------------------------');
    print('总计: ¥${calculateTotal()}');
    print('==============================');
  }
}

void main() {
  var cart = ShoppingCart();

  cart.addItem(CartItem('iPhone', 6999, 1));
  cart.addItem(CartItem('AirPods', 1299, 2));
  cart.addItem(CartItem('充电器', 149, 1));

  cart.printReceipt();
}
```

### 5.3 游戏状态机

```dart
enum GameState { menu, playing, paused, gameOver }

class Game {
  GameState _state = GameState.menu;
  int score = 0;
  int lives = 3;

  GameState get state => _state;

  void start() {
    if (_state == GameState.menu || _state == GameState.gameOver) {
      _state = GameState.playing;
      score = 0;
      lives = 3;
      print('游戏开始！');
    }
  }

  void pause() {
    if (_state == GameState.playing) {
      _state = GameState.paused;
      print('游戏暂停');
    }
  }

  void resume() {
    if (_state == GameState.paused) {
      _state = GameState.playing;
      print('游戏继续');
    }
  }

  void addScore(int points) {
    if (_state != GameState.playing) return;

    score += points;
    print('获得 $points 分，当前总分: $score');
  }

  void loseLife() {
    if (_state != GameState.playing) return;

    lives--;
    print('失去一条生命，剩余: $lives');

    if (lives <= 0) {
      _state = GameState.gameOver;
      print('游戏结束！最终得分: $score');
    }
  }

  void handleInput(String input) {
    switch (_state) {
      case GameState.menu:
        if (input == 'start') start();
        break;

      case GameState.playing:
        switch (input) {
          case 'pause':
            pause();
            break;
          case 'score':
            addScore(10);
            break;
          case 'hit':
            loseLife();
            break;
        }
        break;

      case GameState.paused:
        if (input == 'resume') resume();
        break;

      case GameState.gameOver:
        if (input == 'restart') start();
        break;
    }
  }
}

void main() {
  var game = Game();

  game.handleInput('start'); // 游戏开始！
  game.handleInput('score'); // 获得 10 分
  game.handleInput('score'); // 获得 10 分
  game.handleInput('pause'); // 游戏暂停
  game.handleInput('score'); // 无输出（暂停状态）
  game.handleInput('resume'); // 游戏继续
  game.handleInput('hit'); // 失去生命
  game.handleInput('hit'); // 失去生命
  game.handleInput('hit'); // 失去生命，游戏结束
}
```

## 第6章 最佳实践

1. **优先使用 for-in 而非索引循环**
   
   ```dart
   // 不推荐
   for (int i = 0; i < list.length; i++) {
     print(list[i]);
   }
   
   // 推荐
   for (var item in list) {
     print(item);
   }
   ```
   
2. **使用早期返回减少嵌套**
   ```dart
   // 不推荐：深层嵌套
   void process(User? user) {
     if (user != null) {
       if (user.isActive) {
         // 处理逻辑
       }
     }
   }
   
   // 推荐：早期返回
   void process(User? user) {
     if (user == null) return;
     if (!user.isActive) return;
     // 处理逻辑
   }
   ```

3. **switch-case 要处理所有情况**
   ```dart
   // 使用枚举时确保处理所有 case
   switch (status) {
     case Status.pending:
       return '待处理';
     case Status.completed:
       return '已完成';
     case Status.cancelled:
       return '已取消';
   }
   ```

4. **合理使用断言**
   - 用于验证开发时的假设
   - 不用于处理运行时错误
   - 提供清晰的错误信息
